# IF Food

O IF Food é um sistema completo de delivery de comida que conecta restaurantes, usuários e entregadores em uma única plataforma eficiente e intuitiva.

**Funcionalidades Principais:**

**Restaurantes:**
- Cadastro e gestão de produtos
- Solicitação e acompanhamento de entregas
- Criação e gerenciamento de cupons e promoções
- Atualização do status dos pedidos em tempo real
- Suporte multiusuário para diferentes níveis de acesso
- Geração de relatórios detalhados
- Dashboard com informações relevantes do negócio
- Histórico completo de pedidos
- Comunicação direta com clientes e entregadores
- Sistema de fidelidade para engajar clientes
- Controle de estoque eficiente
- Gestão de pagamentos da plataforma
- Opção para restaurantes com entrega própria
- Funcionalidade de modo pausa para gerenciar a disponibilidade

**Usuários:**
- Cadastro e perfil personalizado
- Navegação intuitiva para realizar compras
- Avaliação de restaurantes e pedidos
- Agendamento de pedidos para conveniência
- Favoritos para acesso rápido aos restaurantes preferidos
- Histórico de todos os pedidos realizados
- Carrinho de compras completo
- Comunicação direta com os estabelecimentos
- Programa de cashback para recompensar a lealdade
- Filtros avançados para encontrar o que procuram

**Entregadores:**
- Cadastro e perfil
- Comunicação direta com os estabelecimentos
- Histórico de todas as entregas realizadas
- Dashboard com informações importantes
- Gestão de pagamentos de suas entregas

**Admin:**
- Cadastro e gerenciamento de cupons promocionais
- Validação de novos cadastros na plataforma
- Dashboard abrangente do sistema
